#О проекте
**BEAVER_KIT** — это многофункциональная OSINT‑платформа для сбора информации с открытых источников   
Собственная нейросеть **Bober AI**, ShadowSocks,DPI‑обход, система авторизации, парсеры вк.

# Возможности

### ГОС-OSINT - РУЧНОЙ
Поиск по открытым источникам:
- **ФИО** — ФССП (долги), ФНС (дисквалификации), КАД (арбитраж), банкротства
- **ИНН** — ФНС, Контур.Фокус
- **Телефон** — оператор, регион, привязка к VK
- **Компания** — реестры юрлиц, арбитраж
- **Госномер авто** — штрафы ГИБДД, ограничения, залоги

### OSINT Beta
Расширенный поиск:
- **Никнейм** — 26+ платформ (GitHub, Twitter, Instagram, Reddit...)
- **VK ID/ник** — профиль, подписчики, группы, фото
- **VK по ФИО** — поиск людей (нужен токен VK)
- **Email** — Gravatar, GitHub, тип домена, утечки (HIBP)
- **IP** — геолокация, провайдер, VPN-флаг, карта

### Сканер ip
- DNS записи, открытые порты, HTTP-заголовки
- WHOIS, SSL-сертификат
- OCR — извлечь текст с фото (нужен `pytesseract`)
- Расшифровка хешей MD5/SHA1/SHA256

### проверка-Авто
Проверка машины по госномеру через ГИБДД — штрафы, ДТП, залоги ФНП.

###  Пасты
Создание HTML-паст/отчеты сотрудников. 8 стилей: Классика, Dark, Orange, Хакер (матрица), Бобёр, Arctic, Пират, Матрица. Экспорт в HTML.

### Склад паст
Хранилище созданных паст — просмотр, открытие, удаление.

### Антималварь
Анализ любого файла:
- Хеши MD5 и SHA256
- Энтропия (высокая > 7.2 → возможно упакован/зашифрован)
- Паттерны малвари: `exec()`, `eval()`, работа с реестром, системные вызовы
- Ссылка на VirusTotal по SHA256
- AI-интерпретация результатов

### AI-ассистент 
Встроенный **Bober AI** — работает полностью офлайн:
- Отвечает на вопросы про BeaverKit и OSINT
- Считает математику
- Показывает курс валют (USD, EUR, BTC, ETH...)
- OCR — загрузи фото и задай вопрос по тексту
- Режим агрессии 

Опционально: подключи **Claude API** (Anthropic) для умных ответов.

### 🦫 Bober Food 
Мощный OSINT на базе SpiderFoot:
- Полный скан домена/IP/email/username
- Субдомены, SSL, Pastebin-утечки, связанные IP
- Установка SpiderFoot одной кнопкой

###  AI-ассистент 
Встроенный **Barsuk AI** — умный Ai осинт специалист 
- Отвечает на вопросы про BeaverKit и OSINT
- может помочь с проверкой уязвимостей 
- может парсить сайты 
- имеет разум логику и свои нормы 
- OCR — загрузи фото и задай вопрос по тексту
- может создать HTML отчет

### 🔒 Beaver VPN — Обходы 
Клиентские методы обхода блокировок (без VPN-серверов):

| Метод | Описание |
|-------|----------|
|   DPI-обход | zapret / byedpi — разбивает пакеты так чтобы оператор не видел       |
|   SNI-подмена | Маскировка под VK/Cloudflare обход белых списков                   |
|   DNS-over-HTTPS | Cloudflare, Google, AdGuard обход простеньких блокировок        |
|   HOSTS-редактор | Быстрый обход через IP не очень эфективен против блокировок     |
|🧅 Tor | Полная анонимность через 3 узла МАХ уровень обхода рабоатет все но медлено |

## VK токен (для OSINT по ФИО)

1. Открой [vkhost.github.io](https://vkhost.github.io/)
2. Нажми VK → Разрешить
3. Скопируй `access_token` из адресной строки
4. Вставь в OSINT Beta → поле «VK токен»
5. можешь спокойно искать цель осинта

## Структура проекта

```
main910.py          # Весь проект —  (~17600 строк)
README.md           # рилми :3
requirements.txt    # Зависимости Python
install.bat         # Установщик Windows
install.sh          # Установщик Linux/macOS
```

Всё написано вручную на PyQt6. Иконки и персонажи (бобёр, лиса, барсук, ёж...) нарисованы кодом через QPainter — никаких внешних картинок.
